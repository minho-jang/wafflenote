**와플노트**는 실시간 화상 강의와 함께 실행하는 **크롬 확장 프로그램**입니다.
> SW마에스트로 11기에서 진행한 NewWave팀 프로젝트 입니다. 

<img src=./imgs/wafflenote1.png>
